# Smart Fit Recipes

Курсова робота — веб-додаток спортивного харчування на Django.

## Паролі

| Користувач | Логін | Пароль | Роль |
|---|---|---|---|
| Адмін | `admin` | `admin` | Суперюзер (адмін-панель) |
| Звичайний юзер | `Nadiya` | *(заданий при реєстрації)* | Користувач |

**Адмін-панель:** http://127.0.0.1:8000/admin/

## Запуск

```bash
source venv/bin/activate
python3 manage.py runserver
```
